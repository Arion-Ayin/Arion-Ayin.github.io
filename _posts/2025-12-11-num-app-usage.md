수비학 계산기(Numerology calculator) 어플리케이션 사용방법에 대한 안내입니다.

---

## 🔢 수비학 계산기는 무엇인가요?

> **수비학 계산기를 사용하여 복잡한 수비학 공식을 쉽고 빠르게 계산할 수 있습니다.**
>
> **인생여정수, 운명수, 혼의수, 성격수, 완성수, 생일수**와 같은 고정적인 수뿐만 아니라,
> 매년, 매달 바뀌는 **1년수, 1달수**까지 한눈에 확인할 수 있습니다.

---

### 1. 수비학의 수와 그 의미

수비학에서 다루는 주요 숫자의 의미는 다음과 같습니다.

| 종류 | 설명 |
| :--- | :--- |
| **인생여정 수** | 나에게 가장 잘 맞는 진로와 직업, 인생의 전반적인 방향성 |
| **운명 수** | 현생에 도달해야 할 궁극적인 목표 |
| **혼의 수** | 내면 깊은 곳에 숨겨진 진짜 욕망 |
| **성격 수** | 타인에게 비춰지는 나, 나의 첫인상과 외적 이미지 |
| **완성 수** | 60세 이후, 인생의 후반기에 완성해야 할 에너지 |
| **생일 수** | 내가 타고난 장점, 가장 잘하고 익숙한 부분 |

| **1년 수** | *올해* 집중해야 하는 분야와 흐름 |
| **1달 수** | *이번 달*에 집중해야 하는 분야와 흐름 |

<br>

### 2. 숫자의 키워드 (긍정/부정)

각 숫자가 가지는 고유한 에너지와 키워드입니다.

| 숫자 | 긍정적 키워드 | 부정적 키워드 |
| :---: | :--- | :--- |
| **1** | 결심, 시작, 출발, 의지 | 성급함, 충동적, 일시적, 독단적 |
| **2** | 조화, 균형, 대칭, 아름다움 | 의존, 질투, 갈등, 흑백논리 |
| **3** | 자기표현, 자신감, 즐거움, 유희 | 가벼움, 산만함, 과장 |
| **4** | 안전, 보호, 책임감, 조직, 관리 | 비판적, 고집불통, 변덕 |
| **5** | 변화, 경험, 정보, 다양성, 자유 | 불안정, 무규칙, 산만함 |
| **6** | 이해, 통합, 헌신, 가족, 사랑 | 간섭, 집착, 무책임, 비상식 |
| **7** | 이상, 영성, 꿈, 비전, 자아실현 | 현실도피, 과잉보호, 인정욕구 |
| **8** | 노력, 성실함, 현실적인 힘, 자수성가 | 물질만능, 힘의 과시, 비현실성 |
| **9** | 집중, 기다림, 은둔, 이타심, 완성 | 복수심, 냉소적, 과거 집착 |
| **0** | 무한한 가능성, 아직 형태가 없는 잠재력 | 혼돈, 공허함 |

**사용법:**
*   **이름 입력**: 분석할 대상의 이름을 입력합니다.
*   **생년월일 선택**: 달력 아이콘을 눌러 정확한 생년월일을 선택합니다.
*   **계산하기**: 입력 후 **계산 버튼**을 누르면 즉시 분석 결과가 나타납니다.
